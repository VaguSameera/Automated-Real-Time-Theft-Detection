PROJECT ANALYSIS

Overview

Shield AI is a real-time surveillance solution designed to mitigate retail loss by identifying suspicious shoplifting behaviors. Utilizing the YOLOv8 Nano model, the system processes live video feeds to detect Class 1 (Shoplifting) and Class 0 (Normal) behaviors with high precision.

System Architecture

AI Engine

  • Model: YOLOv8 (You Only Look Once)
  • Framework: Ultralytics & PyTorch
  • Classes: Normal Behavior vs. Shoplifting
  • Inference: Real-time frame analysis

Backend Architecture

  • Server: Flask (Python)
  • Streaming: Multipart Mixed-Replace
  • Image Processing: OpenCV & imutils
  • Data Handling: Persistent Session Logging

Detection Logic

The video stream is optimized to a width of 500px to ensure low-latency processing. The system skips every 2 frames to maintain 30FPS throughput on standard hardware. When a threat is confirmed with a confidence score above 40%, the system captures a high-resolution screenshot and logs the timestamp to the "Detect" history.
